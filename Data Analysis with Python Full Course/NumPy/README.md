- Dimensions & Shapes
- Indexing & Slicing of Matrices
- Summary Statistics
- Broadcasting & Vectorised Operations
- Boolean Arrays
- Linear Algebra
- Size of Objects in Memory
- Large Lists 
- Useful NumPy Functions
